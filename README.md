# OLA-Data-Analysis-Project

•	This project shows crucial data analytics skills and understands the workflow of a complete data project, from raw data to compelling visualization. Excel for deeper analysis, from data sorting and pivot tables to advanced formulas, it discovers how to perform meaningful data interpretation and summarization
•	SQL for clean and prepare data efficiently, and handle complex data structures using various aggregate functions
•	Power BI for Dashboard Creation, it shows how to transform insights into impactful visual stories and create an interactive and visually appealing dashboard that communicates key findings effectively using DAX
